JDisc is Vespa's Java-container, optimized for a quick and easy dev-to-prod cycle. It is fully integrated with the Cloud Config System, allowing for live reconfiguration without service interruption. The container exposes a servlet-like API for query processing that allows you to build RESTful web-services, and a storm-like API to do document processing. It exposes APIs for simple and performant federation across heterogeneous data sources.

  • Develop RESTful APIs using Jersey.

  • JDisc provides a framework for processing requests and responses, aptly named "Processing". Its building blocks are:
    • Chains of other components that are to be executed serially, with each providing some service or transform.
    • Processors that change the request and / or the response. They may also make multiple forward requests, in series or parallel, or manufacture the response content themselves.
    • Renderers that are used to serialize a Processor's response before returning it to a client.
    • Use case: Building an HTTP API using request handlers and processors
