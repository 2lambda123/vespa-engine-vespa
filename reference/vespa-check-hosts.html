This is the HTML version of the man page for the command line tool vespa-check-hosts — a tool you can use to check the hosts in your setup for OS version, kernel version, valid hostnames, valid disk setup.

Name

vespa-check-hosts

Synopsis

vespa-check-hosts

Description

vespa-check-hosts is a command line tool you can use to check valid setup for all nodes in a Vespa installation. It needs a running cloudconfig_server or a running services package on the node you are executing the script on to work.

Options

None

Examples

$ vespa-check-hosts
no services found with type 'storagenode'
WARNING: [host-url] has kernel 2.6.9-023stab048.4-smp, Vespa recommends at least 2.6.9-89 (this is RHEL 4u8)
Checking search hosts
