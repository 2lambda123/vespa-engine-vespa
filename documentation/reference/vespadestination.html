This document describes vespadestination.

vespadestination

vespadestination is a small java program, similar to vespavisittarget in that it can receive messages from messagebus and print the contents to stdout. It can be useful in situations where you want to debug a route or a docproc, by using the vespadestination as the endpoint of your route.
