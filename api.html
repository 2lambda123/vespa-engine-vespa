Data APIs

  • Feeding API: High performance feeding API, the recommended API for feeding data to Vespa
  • RESTified Document Operation API: Simple REST API for operations based on document ID (get, put, remove, update)
  • JSON feed format: The Vespa Document format
  • DEPRECATED: Simple HTTP API: A simplistic HTTP API for POST'ing feed files to Vespa. The API also supports GET,REMOVE and VISIT operations
