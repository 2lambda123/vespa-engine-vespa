Vespa

The open big data serving engine.

Get started with Vespa

Vespa allows you to easily serve from your big data - in milliseconds at any scale and with any traffic volume. Use it to:

  • Match items using structured sql-like queries and unstructured search.
  • Organize and process all matches to generate navigation pages, tag clouds, graphs and so on on the fly.
  • Rank matches by executing handwritten or machine-learned relevance models.
  • Serve data and results.
  • Write data in real time while serving at a rate of thousands per second per node.
  • Grow, shrink and reconfigure clusters while serving and writing.

    Use it to build

    • Search applications
    • Personalized recommendation
    • Navigation pages computed on demand
    • Realtime data displays
    • Combinations of the above
