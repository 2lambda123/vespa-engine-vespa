Vespa

The open big data serving engine

Get started with Vespa

Vespa allows you to easily serve from your big data - in milliseconds at any scale and with any traffic volume.

Vespa let's you

  • Match with sql-like queries and unstructured search
  • Organize all matches to generate data-driven pages
  • Rank matches by handwritten or machine-learned relevance models
  • Write data in real time thousands of times per second per node
  • Grow, shrink and reconfigure clusters while serving and writing

Use Vespa to build

  • Search applications
  • Personalized recommendation systems
  • Navigation pages computed on demand
  • Realtime data displays - tag clouds, maps, graphs
  • Combinations of the above
